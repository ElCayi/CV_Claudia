Actualmente estoy en proceso de desarrollo de las habilidades técnicas necesarias para convertirme en desarrolladora web, centrándome en las tecnologías más demandadas en el mercado gallego como React, Java/Spring Framework, y sistemas de gestión de contenidos.

Mi Enfoque de Aprendizaje

Como profesional con perfil INTJ-Arquitecta, mi enfoque de aprendizaje es estructurado y estratégico. Estoy aprovechando las competencias adquiridas en mi formación DAW y priorizando el desarrollo de habilidades en:

Frontend

  • HTML5, CSS3 y JavaScript avanzado
  • Frameworks como React
  • Diseño responsive y accesibilidad web
  • Transformación y presentación de datos
  • Buenas prácticas semánticas

Backend

  • Java y Spring Framework
  • Programación orientada a objetos
  • Desarrollo de APIs REST
  • Diseño y gestión de bases de datos SQL
  • Arquitectura de sistemas y microservicios

Competencias técnicas y metodológicas

  • Sistemas informáticos y configuración de redes
  • Diagramas de clases y pruebas funcionales
  • Gestión de versiones con Git
  • Rendimiento y seguridad en aplicaciones
  • Documentación profesional de código
  • Diseño modular y patrones de arquitectura
  • Uso profesional de entornos IDE
  • Metodologías ágiles de desarrollo

Mi objetivo es convertir este aprendizaje teórico en proyectos prácticos que demuestren mis capacidades y me permitan insertarme en el mercado laboral gallego, aplicando tanto mis conocimientos técnicos como mi capacidad analítica.
